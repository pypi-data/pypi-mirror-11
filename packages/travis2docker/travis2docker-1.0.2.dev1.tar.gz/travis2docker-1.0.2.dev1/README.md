# travis2docker

Script to generate Dockerfile from .travis.yml file

## Install
`# pip install git+https://github.com/Vauxoo/travis2docker.git`

## Usage
 `travisfile2dockerfile REPO_URL BRANCH`

 For more information execute:
 `travisfile2dockerfile --help`
